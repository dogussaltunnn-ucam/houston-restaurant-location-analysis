# Houston Restaurant Location Analysis

This project analyzes restaurant distribution across the Houston area using SQL and Tableau. The goal was to explore geographic concentration, city-level restaurant counts, price range patterns, and the most common cuisine tags.

## Tools Used
- SQL (DBeaver / SQLite)
- Tableau Public
- GitHub

## Project Objective
The objective of this project was to analyze restaurant locations in the Houston area and build a dashboard that highlights:
- restaurant distribution on a map
- top cities by restaurant count
- price range distribution
- most common cuisine tags

## Dataset
The dataset includes restaurant-level location and descriptive information such as market, city, restaurant name, category description, price range, latitude, and longitude.

## SQL Analysis
The SQL analysis focused on:
- total restaurant count
- restaurant counts by city
- price range distribution
- extraction of common cuisine tags
- map-ready export with latitude and longitude

## Key Insights
- Houston dominates the dataset with the highest restaurant concentration.
- Katy, Sugar Land, Cypress, and Spring follow as secondary restaurant clusters.
- The most common price range in the dataset is `$$`.
- Higher-end `$$$$` restaurants are extremely limited.
- The most common cuisine tags include Mexican/Tacos, Italian/Pizza, American/Burgers, and Japanese/Sushi.
